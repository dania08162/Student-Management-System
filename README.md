# Student-Management-System
A python-based project about managing students in an effective and useful way.

## Purpose
I built this project to practice programming and to support my work teaching students!

## Features
- Adding/removing students
- Adding & updating student grades
- Viewing student profile from a student, parent, teacher, or admin point of view

## Technologies Used
- Python
- Google Colab

## What I Learned
I learned more about using dictionaries, handling user input & data, writing functions using loops, and how to organize code effectively.
